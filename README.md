# Multi-Language Web Development Showcase

A comprehensive web application demonstrating proficiency in multiple programming languages and technologies for web development.

NB: Worked on this project with some help from Gemini 2.5 Flash, ChatGPt & Cursor.

## 🚀 Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: PHP, Python (Flask), Ruby (Rails), C# (ASP.NET), Perl
- **Database**: MySQL, SQLite
- **Additional**: Chart.js, Bootstrap, jQuery

## 🛠️ Setup Instructions

### Prerequisites
- Web server (Apache/Nginx)
- PHP 7.4+
- Python 3.8+
- Ruby 2.7+
- .NET Core 3.1+
- Perl 5.30+
- MySQL 8.0+

### Quick Start
1. Clone this repository
2. Set up your web server to point to the project directory
3. Install dependencies for each language:
   - PHP: No additional setup required
   - Python: `pip install -r python/requirements.txt`
   - Ruby: `bundle install` in the ruby directory
   - C#: Open solution in Visual Studio
   - Perl: Install required modules via CPAN
