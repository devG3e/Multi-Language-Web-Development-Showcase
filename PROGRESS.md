## 🔧 Setup Instructions

### Prerequisites
- Web server (Apache/Nginx) with PHP support
- Python 3.8+
- MySQL 8.0+
- Node.js (for development tools)

### Quick Start
1. Clone the repository
2. Run `python setup.py` for automated setup
3. Configure database settings in `php/config.php`
4. Start Python Flask app: `cd python && python app.py`
5. Access via web server for PHP components
